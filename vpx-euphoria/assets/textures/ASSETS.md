# Textures

All textures should be PNG, power-of-two dimensions (e.g. 2048×2048 for playfield).

| File | Role | Notes |
|------|------|-------|
| `playfield_diffuse.png` | Main playfield color map | Slice from generated playfield art |
| `playfield_plastic.png` | Plastic overlay / inserts | Transparent PNG, alpha for emission |
| `bumper_cap_pink.png` | Bumper cap texture | Hot pink (#FF1493) base, glitter overlay |
| `bumper_ring_cyan.png` | Bumper skirt ring | Cyan (#00FFFF) neon ring |
| `ferris_diffuse.png` | Ferris Wheel color map | Carnival color scheme |
| `ferris_normal.png` | Ferris Wheel normal map | |
| `door_diffuse.png` | Bathroom Door surface | Grunge + title card graphic |
| `dt_M.png` `dt_A.png` `dt_D.png` `dt_Y.png` | Drop target faces | One per letter, 128×256 |
| `door_target.png` | Bathroom Door hit face | |

## Color palette reference

| Role | Hex |
|------|-----|
| Primary / Purple | `#800080` |
| Secondary / Cyan | `#00FFFF` |
| Accent / Glitter Pink | `#FF1493` |
| Background dark | `#0a0a14` |
