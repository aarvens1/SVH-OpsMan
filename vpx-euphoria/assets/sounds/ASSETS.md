# Sounds

All sounds must be **32-bit PCM WAV, 44100 Hz** for VPX Sound Manager compatibility.

## Music tracks (slice from Labrinth Euphoria OST)

| File | Source track | Loop point notes |
|------|-------------|-----------------|
| `attract_loop.wav` | Labrinth — "Intro" | Loop at 8-bar boundary, ~128 BPM |
| `music_cassie_mode.wav` | "Cassie" instrumental | 20s version for hurry-up |
| `music_party_mode.wav` | "Jacob's Party" | Upbeat section, 45s loop |
| `music_fezco_mode.wav` | "Fezco" cue | Tense, 45s loop |
| `all_for_us_intro.wav` | "All For Us" — intro sting | 4-bar build |
| `all_for_us_full.wav` | "All For Us" — full | 60s loop for wizard mode |

**Tempo:** ~128 BPM — `all_for_us_full.wav` beat subdivisions drive BPM-synced flashers in wizard mode.
